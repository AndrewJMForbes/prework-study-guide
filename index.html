HTML

  • The !DOCTYPE declaration tells the browser what document type should be expected when opening the webpage.
  • The head element contains information about the webpage.
  • The head also contains the title element which defines the title for the webpage. It is extremely important for Search Engine Optimization (SEO).
  • The script element embeds client-side scripts in the HTML document it is also located in the head element.
  • The lang attribute tells the webpage what language it should be dispayed in.
  • The meta element is found inside the head element and contains information such as character set (charset), descriptions, keywords, author, and viewport.
  • The style element can also be found in the head element and it defines the CSS styles attached with the webpage.
  • The body element represents the visible content shown to the user.
  • The header tag appears at the top of the page and usually contains a navigation menu and/or intoductory content.
  • Header tags range from H1 to H6.
  • The p element represents a paragraph tag.
  • There are 3 types of list tags ul, ol, and li>.
  • The img tag contains an image displayed on the webpage.
  • A div creates a division in a page.
  • A span creates an inline division on the webpage.
  • There are elements that create a line of empty space between two blocks of content using the br tag.
  • Hyperlink elements whether linked to the same webpage or other pages will use the a tag (which stands for anchor).
  • Accompanying the a tag is the href attribute which directs you to the link attached to the tag.
  • The footer appears at the bottom of the page and usually contains information such as the author, copyright, contact, sitemap, and navigation.

CSS

  • CSS stands for Cascading Style Sheets.
  • CSS is a programming language used to define how webpage content is presented to users.
  • The style element inside the head element contains CSS rules for the webpage which embeds an entire stylesheet directly in the HTML file.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • The box shadow property adds shadow effects around an element's frame. You can set multiple effects separated by commas. A box shadow is described by X and Y offsets relative to the element, blur and spread radius, and color.
  • The link element creates a connection to the external file.
  • The rel attribute specifies the relarionship between the current document and the linked document resource.
  • The href attribute specifie the location (URL) of the external resource.

Git

  • The main branch is where all changes eventually get merged back into also referred to as master.
  • The git command must precede every Git statement.
  • The add command adds modifications in the working branch.
  • The -A flag indicates that we wat to add all changes.
  • The git commit command captures a snapshot of the repo at a specific point in time.
  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • The checkout command is to move the working branch to a new branch
  • The -b flag creates a new branch
  • The pull command is used to receive a branch's modifications in the local environment.
  • The push command can create a new remote branch or add a new changes to an existing branch.
  • The origin keyword indicates the source of the pull which would be in the GitHub repository (repo).

JavaScript

  • JavaScript is a high-level dynamic programming language that allows users to interact with websites.
  • Control flow is the order in which the computer exectutes statemnets in a script. Code is run in order from the first line in the file to the last line, unless the computer runs across the structures that change the control flow.
  • In JavaScript we declare by using the "var" keyword, then we give the variable a name because it will reference the information stored in the variable in our code.
  • A string is a sequence of one or more characters that may consist of letter, numbers, or symbols. Strings in JavaScript are immutable, which means they are unchanging. All strings are surrounded by quotation marks.
  • A console.log method outputs a message to the web console.
  • Conditional statements in JavaScript such as "if" introduce decision-making into our code.
  • If a computer encounters an "if" statement in JavaScript it will check to see if the condition that we provide is truthy. If the condition isn't true or has a value that isn't truthy, then the computer will ignore whatever is between the curly brackets and move on.
  • Truthy & falsy- A value is considered falsyif it one of the following values: 0, -0, On, "", null, undefined, NaN, or a Boolean data type with the value of a false. All other values, including a Boolean with a value of true are considered truthy.
  • A for loop executes a block of code as long as a specified condition is true. JavaScript for loops take three arguments: initialization, condition, and increment.
